# Analogía del mundo real

## Imagen de Referencia
![Diagrama](https://refactoring.guru/images/patterns/diagrams/template-method/live-example.png?id=2485d52852f87da06c9cc0e2fd257d6a)

Un plano arquitectónico típico se puede modificar ligeramente para adaptarse mejor a las necesidades del cliente.

El método de plantillas se puede utilizar en la construcción de viviendas en masa. El plano arquitectónico para construir una casa estándar puede contener varias ampliaciones que permitirían al posible propietario ajustar algunos detalles de la casa resultante.

Cada paso de la construcción, como colocar los cimientos, enmarcar, levantar las paredes, instalar las tuberías y el cableado de agua y electricidad, etc., se puede modificar ligeramente para que la casa resultante sea un poco diferente de las demás.
